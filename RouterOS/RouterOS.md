
Expose address
```
ip address add interface=ether2 address=192.168.56.11/24
```

Remove address

```
ip address remove 0
```

Print IP address
```
ip address print
ip ad pr
```

Print interfaces
```
interface print
in pr
```

Check updates
```
system package update check-for-updates
```

Reboot
```
system reboot
```
